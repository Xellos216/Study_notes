# Git 연습 프로젝트

이 저장소는 Git과 GitHub의 기초 개념을 실습하고 익히기 위한 연습용 프로젝트입니다.

## ✅ 실습 내용

- Git 저장소 초기화 (`git init`)
- 커밋 흐름 실습 (`add`, `commit`, `push`)
- GitHub 원격 저장소 생성 및 연동
- `.gitignore` 사용하여 민감 파일 제외
- Markdown을 활용한 문서 작성 연습

## 📂 디렉토리 구조 예시
git-practice/
├── .gitignore
├── README.md
└── mysecret.txt (→ Git에서 제외됨)


## 🔧 사용 기술

- Git CLI
- GitHub
- Markdown
